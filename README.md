# Jap_bacuram: Gemini CLI Skills Collection

このリポジトリは、Gemini CLIのエージェント性能を拡張するための高度なスキルセットを公開しています。

## 公開中のスキル

### 1. 反証可能性チェック (falsifiability-check)
提示された文章の論理構造を抽出し、「反証可能性チェックリスト」に基づき、挑戦・代案・阻止・反駁の4つの観点から論理の脆弱性を徹底的に検証します。

#### 特徴
- **論理構造の可視化**: 主張（C）と根拠（R）を明確に分解。
- **厳格な強度試験**: 独自の「汎用判定アルゴリズム」により、論理の「誠実さ（負の側面への向き合い）」を客観的に評価。
- **反論の先回り**: トレードオフや代案の欠落を指摘し、より堅牢な議論へのブラッシュアップを支援。

#### 判定アルゴリズム（共通基準）
- **✅ 問題なし（堅牢）**: 主張に伴うコストやリスクが明示され、合理的な理由や緩和策が示されている。
- **⚠️ 懸念あり（部分的欠落）**: 負の側面の評価が曖昧、または抽象的な表現で回避されている。
- **❌ 脆弱性あり（盲点）**: メリットのみを強調し、確実な負の側面や制約を無視している。

## インストール方法

Gemini CLIがインストールされている環境で、以下のコマンドを実行してください：

```bash
gemini skill install https://github.com/Mins-sr/Jap_bacuram/tree/main/falsifiability-check
```
