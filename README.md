# Jap_bacuram

「Jap_bacuram」は、Gemini CLIのエージェントを強化するスキルをまとめたリポジトリです。

### 反証可能性チェック (falsifiability-check)
文章の論理をバラバラにして、その脆さをあぶり出すスキルです。

このスキルは、文章の「言い分」と「その理由」を分け、「挑戦」「代案」「阻止」「反駁」の4つの角度から厳しく問い直します。単なる要約ではなく、書き手が隠している「不都合な事実」や「無視されたコスト」を特定します。

#### 判定のものさし
判定には「汎用判定アルゴリズム」を使い、論理の誠実さを評価します。

*   ✅ **問題なし（堅牢）**
    「デメリット」や「リスク」を正直に認め、その上で納得できる理由や対策が語られている。
*   ⚠️ **懸念あり（部分的欠落）**
    悪い面への言及が曖昧だったり、精神論で誤魔化したりしている。反論の隙がある。
*   ❌ **脆弱性あり（盲点）**
    いいことしか言っていない。現実の「コスト」や「制約」を無視しており、実行時に破綻する可能性が高い。

### 導入方法
Gemini CLIがインストールされた環境で、以下のコマンドを実行してください。

```bash
gemini skill install https://github.com/Mins-sr/Jap_bacuram/tree/main/falsifiability-check
```
